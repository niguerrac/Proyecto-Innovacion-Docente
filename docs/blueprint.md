# **App Name**: Tikap Skills

## Core Features:

- User Registration: User registration and profile creation.
- Skill Assessment: Skill assessment quiz with weighted questions to identify user skill gaps.
- Content Recommendation: AI tool to Recommend personalized learning modules based on assessment results, highlighting areas for improvement. LLM will use tool to include modules with the content user is lacking.
- Module Catalog: Display a catalog of available learning modules, searchable and filterable by topic.
- Progress Tracking: Progress tracking system to show users their advancement in each module, using visual indicators like progress bars.

## Style Guidelines:

- Primary color: FB2331 for main interactive elements
- Contrast color: F0F757 for accents and highlights.
- Details color: FB2331 to match main interactive elements
- Font: 'Lenxend' (sans-serif) for headings and body text to give personality
- Use simple, clear icons to represent different skill categories and module types.
- A clean and intuitive layout with clear visual hierarchy.
- Use subtle transitions and animations to enhance user experience.
